My projects:

Wether-map

It is a weather application, to use it, just enter the city you are interested in in the search field, click search and you can already see the weather for a given city. The weather is additionally represented on the map so that you can check where the location is. The application is still waiting for an improvement to the graphic design, which I will do soon.

Technologies:

  • Bootstrap
  • Thymeleaf
  • OpenWeather
  • OpenStreetMap

Car-managment

It is an application, two to be precise. The first is a rest application built on java and springboot, it displays data about cars such as: make, model and color. The second, completely separate application, built in angular, listens to the first and presents it graphically. You can add new car items, delete existing ones or edit them.

Technologies:

  • TypeScript
  • Java
  • SpringBoot
